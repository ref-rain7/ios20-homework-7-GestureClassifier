在 APP 的最下方显示当前预测的姿势及其 confidence.

按下“Start Game”按钮开始游戏，在每个回合，用户需要在 3 秒内完成随机指定的动作，成功完成则计入“✔︎”，超时则计入“✘”。按下“End Game” 按钮来结束游戏。
